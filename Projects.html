AARUHI_BOT - Project Overview

This project involves creating a chatbot named "AARUHI_BOT" using the Rasa framework and the NASA API key. The bot fetches and provides information about NASA's Image of the Day, making space exploration more accessible to users.

Technologies Used

Python, Rasa, NASA API, HTML, CSS

Challenges Faced

Integrating the NASA API with the Rasa framework and ensuring accurate information retrieval were some of the challenges I overcame during this project.

Learning Outcome

I enhanced my skills in API integration, chatbot development, and Python programming while working on this project.
